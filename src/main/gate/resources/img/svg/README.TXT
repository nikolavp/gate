The SVG images in this folder were converted into Java2D nased classes using
SVGRoundTrip: http://englishjavadrinker.blogspot.com/search/label/SVGRoundTrip

The following command line arguments were used to control the conversion

	-n +Icon
	-o src
	-p gate.resources.img.svg
	-t scalable
	src/gate/resources/img/svg
